Aperio Design System Specification1. Core PrinciplesGlassmorphism: Use translucent backgrounds with backdrop-filter: blur().Living Interface: Micro-interactions and subtle animations (float, shine, pulse) are mandatory for all interactive states.Theme Consistency: All components must support theme-dark, theme-light, and theme-magic via global body class switching.2. Component ArchitectureA. Sidebar (Floating Glass Dock)Structure: Fixed positioning, vertical pill-shape (rounded-[44px]).State: Use .nav-box for items. Use .active class for selected state (applies background glow + indicator).Indicators: Use .nav-indicator (a 4px wide vertical strip on the left).Tooltip: Apply .nav-tooltip (Absolute position, left-aligned, fade-in on hover).B. Glass Cards (.glass-panel)Class requirements: backdrop-blur-xl, border, rounded-[32px].Interaction: Must have .glass-shine for the sweeping light effect.Hover: Apply .glass-panel-hover for lifting effect (translate-y-2) and enhanced shadow.C. Transaction Items (.tx-item)Action Buttons: Hidden by default. Show only on group-hover (opacity-0 -> opacity-100).Buttons: Edit (Blue theme), Delete (Red theme). Use standard Tailwind transition-all.4. UI Library Guidelines for AIWhenever the AI refactors a component, it must strictly follow these rules:State Preservation: Keep all props, useState, useEffect, useQuery unchanged.Markup Conversion:Replace standard div backgrounds with .glass-panel.Ensure all text has color classes that are theme-aware (use .keep-white for text that must remain white regardless of theme).SVG Handling: * Use the classes .svg-dark, .svg-light, .svg-magic to swap icon color variants dynamically.Ensure icon paths reference the CSS variables (var(--ic-dash-1), etc.) for seamless theme transitions.No Logic Leakage: CSS classes must not interfere with functional attributes (onClick, id, ref).5. Animation Token RegistryFloat: animate-float (Y-axis oscillation).Gradient Flow: animate-gradient-x (For background shifting).Shine: animate-[shine_1.5s_ease-in-out] (For buttons/cards).Blob: animate-blob (For background ambient light movement).
